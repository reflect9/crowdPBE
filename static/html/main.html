The purpose of programming is to teach computer how to perform a task. Traditionally, programmers do this by writing out instructions. But as an alternative approach, Programming-by-Example enables people to teach computer with examples of input and output, and then the computer will infer programs that calculate the outputs for the inputs. For instance, to teach computer a program that multiplies any number by five, you can give an example (Input:1, Output:5). Then the computer will infer programs such as Input * 5 that calculates 5 for 1.

In this study, we are going to ask you to teach computer a few tasks by providing examples. If you provide sufficient and thoughtful examples, the computer will effectively infer a single program that performs the task for a wide range of inputs beyond the cases that you provide. Don't worry much now. You will learn how to do it while following tutorials.

You will be asked to provide examples for 6 programs. Please add as many steps and cases as you think is necessary to explain how the program should work. Each task will begin when you click the input-output table. Thus feel free to take a break or to read instruction before clicking the input-output table. Try not to be interrupted while doing each task.
